Software Engineering

Science which includes the development and building of computer systems software and applications software.
What do Software Engineers do?
Software engineers typically do the following:
  • Develop or direct software system testing or validation procedures
  • Direct software programming and development of documentation
  • Analyze information to determine, recommend, and plan installation of a new system
  • Consult with engineering staff to evaluate interface between hardware and software
  • Monitor functioning of equipment to ensure system operates

Software engineering development over history
Software engineering is probably the fastest growing engineering type some the first history bechmarks only go 50 years back!
  • Back in 1946, J. Presper Eckert Finished the first computer. When it was done, it weighed almost 50 tons and programmers didn't interact directly with computing devices.
  • In 1957, the first widely used programming language was released (Fortran)
  • In early 1971, Nutting Associates released the first video game known as "Computer Space".
  • In March of 1973, Xerox released the first desktop computer named the Xerox ALTO.
  • In January of 1983, Robert Kahn and Vint Cerf created a system known as ARPANET which was basically a "Network of Networks" or internet
  • Tim Berners-Lee, a British Scientist from CERN, invented the World Wide Web.
  • In May of 1995, Brendan Eich created JavaScript
  • Invention of supercomputers, smartphones, etc...
